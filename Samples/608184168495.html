If a number is formed of \(2\) different randomly chosen digits from the set \(\{4, 5, 8\}\), find the probability of the units digit being odd.
The sample space is \[S = \{45, 48, 54, 58, 84, 85\} \text.\]
The sample space, \(S\), contains all the possible numbers formed from \(2\) different digits of the set \(\{4, 5, 8\}\).

There are three possibilities for the first digit: \(4\), \(5\), and \(8\).

If the first digit of the number is \(4\), the second digit must be either \(5\) or \(8\), so the possible numbers are \(45\) and \(48\).

If the first digit of the number is \(5\), the second digit must be either \(4\) or \(8\), so the possible numbers are \(54\) and \(58\).

If the first digit of the number is \(8\), the second digit must be either \(4\) or \(5\), so the possible numbers are \(84\) and \(85\).

Hence the sample space, \(S\), is \(\{45, 48, 54, 58, 84, 85\}\).

The event, \(E\), is the set of outcomes of the sample space with odd units digit:

\[E = \{s \in S: s \text{ has odd units digit} \} = \{45, 85\} \text.\]
Hence, \[P\left(\text{odd units digit}\right) = \frac{2}{6} = \frac{1}{3} \text.\]
The probability of an event \(E\) is \[P(E) = \frac{n(E)}{n(S)} \text,\] where \(n(E)\) is the number of outcomes in the event \(E\) and \(n(S)\) is the number of elements in the sample space \(S\).

Hence,

\[\begin{align} P\left(\text{odd units digit}\right) & = \frac{n(\{45, 85\})}{n(\{45, 48, 54, 58, 84, 85\})} \\ & = \frac{2}{6} \\ & = \frac{1}{3} \text. \end{align}\]
